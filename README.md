# Todo list
---

In this program you choose command between 1-7.
- 1 => Adds a task in a list
- 2 => Changes the name of the task
- 3 => Removes specific task from list
- 4 => Shows some information of a specific task
- 5 => Shows all tasks exists in a list
- 6 => Marks a certain task as completed
- 7 => Closes the program
  
It stores all temporary tasks in arraylist.
